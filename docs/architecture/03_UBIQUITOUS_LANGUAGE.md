# Lenguaje Oficial de Instituva

Este documento establece el lenguaje único y oficial de Instituva. Su propósito es asegurar que arquitectura, documentación, desarrollo, interfaces, ayuda contextual, manuales e integraciones futuras utilicen los mismos conceptos con el mismo significado.

Este documento no define arquitectura, procesos, modelos de datos, pantallas ni implementación.

# Colecciones

Definición.
Dominio principal de Instituva responsable de administrar el patrimonio museológico y el expediente oficial de cada pieza.

Uso dentro de Instituva.
Se utiliza para referirse al área que conserva la información oficial sobre piezas, bienes patrimoniales, procedencia, titularidad, custodia, documentación, movimientos, conservación, restauración, préstamos, donaciones e historial museográfico.

Notas.
Colecciones no es sinónimo de Inventario Administrativo.

# Patrimonio Museológico

Definición.
Conjunto de piezas, bienes, objetos, documentos, obras e instrumentos con valor cultural, histórico, artístico, documental, musical, educativo o museográfico bajo responsabilidad del museo.

Uso dentro de Instituva.
Se utiliza para referirse al valor institucional que Colecciones protege, organiza y documenta.

# Pieza

Definición.
Objeto individual con valor museológico que forma parte del patrimonio o está bajo responsabilidad museográfica del museo.

Uso dentro de Instituva.
Se utiliza como término principal para identificar cada objeto gestionado por Colecciones.

Notas.
Cuando el contexto requiera mayor formalidad, puede utilizarse Bien Patrimonial, pero Pieza es el término operativo preferido.

# Bien Patrimonial

Definición.
Pieza reconocida por su valor cultural, histórico, artístico, documental, musical, educativo o museográfico.

Uso dentro de Instituva.
Se utiliza en contextos institucionales o documentales donde se requiera enfatizar el carácter patrimonial de una pieza.

# Expediente Museográfico

Definición.
Registro institucional completo que conserva la memoria oficial de una pieza.

Uso dentro de Instituva.
Se utiliza para referirse al conjunto de información que explica qué es la pieza, de dónde proviene, quién tiene titularidad, bajo qué custodia se encuentra, dónde está, cuál es su condición, qué procesos ha vivido y qué documentación la respalda.

Notas.
Cada pieza tiene un único expediente museográfico oficial.

# Número de Inventario Museográfico

Definición.
Identificador oficial asignado a una pieza dentro del inventario museográfico.

Uso dentro de Instituva.
Se utiliza para localizar, consultar, documentar y relacionar una pieza con su expediente museográfico.

Notas.
No debe confundirse con número de sello, etiqueta de activo o identificación administrativa de equipos operacionales.

# Identificación

Definición.
Conjunto de datos que permite reconocer una pieza de manera clara y distinguirla de otras.

Uso dentro de Instituva.
Se utiliza para agrupar nombre, número de inventario museográfico, descripción, clasificación y cualquier dato necesario para reconocer la pieza.

# Clasificación

Definición.
Categoría museográfica que ubica una pieza dentro de un tipo o grupo conceptual.

Uso dentro de Instituva.
Se utiliza para organizar piezas según su naturaleza, función, material, contexto cultural o criterio museográfico aprobado.

# Catalogación

Definición.
Proceso conceptual de describir, organizar y registrar una pieza dentro del patrimonio museológico.

Uso dentro de Instituva.
Se utiliza para referirse a la formalización museográfica de la información de una pieza dentro de su expediente.

# Procedencia

Definición.
Origen conocido o documentado de una pieza, incluyendo su trayectoria antes de llegar al museo.

Uso dentro de Instituva.
Se utiliza para explicar de dónde proviene una pieza y qué información respalda su relación con el museo.

# Titularidad

Definición.
Condición que indica quién posee legalmente una pieza.

Uso dentro de Instituva.
Se utiliza para distinguir si una pieza pertenece institucionalmente al museo o si pertenece a otra persona o entidad.

Notas.
Titularidad no es lo mismo que adquisición ni custodia.

# Custodia

Definición.
Responsabilidad del museo sobre el cuidado, protección y manejo de una pieza, aunque la titularidad pertenezca a otra persona o entidad.

Uso dentro de Instituva.
Se utiliza cuando el museo conserva, exhibe, protege o administra temporal o permanentemente una pieza bajo su responsabilidad.

# Propiedad Institucional

Definición.
Condición de una pieza cuya titularidad pertenece oficialmente al museo o a la entidad institucional correspondiente.

Uso dentro de Instituva.
Se utiliza para distinguir piezas propias de piezas prestadas, depositadas o bajo custodia temporal.

# Adquisición

Definición.
Forma autorizada mediante la cual una pieza ingresa al patrimonio o responsabilidad museográfica del museo.

Uso dentro de Instituva.
Se utiliza para registrar el origen administrativo o institucional del ingreso de una pieza.

Notas.
La adquisición no define por sí sola la titularidad.

# Compra

Definición.
Forma de adquisición en la que una pieza ingresa mediante una transacción autorizada.

Uso dentro de Instituva.
Se utiliza para identificar piezas incorporadas al museo por adquisición onerosa aprobada.

# Donación

Definición.
Forma de adquisición mediante la cual una persona o entidad entrega una pieza al museo sin requerir pago a cambio.

Uso dentro de Instituva.
Se utiliza como proceso interno del ciclo de vida de Colecciones.

Notas.
Una pieza donada puede convertirse en propiedad institucional cuando el proceso correspondiente queda completado.

# Donante

Definición.
Persona o entidad que entrega una pieza al museo mediante donación.

Uso dentro de Instituva.
Se utiliza para identificar la fuente de una donación y relacionarla con el expediente museográfico correspondiente.

# Préstamo

Definición.
Proceso mediante el cual una pieza entra o sale temporalmente bajo condiciones documentadas, sin que necesariamente cambie su titularidad.

Uso dentro de Instituva.
Se utiliza como proceso interno del ciclo de vida de Colecciones para documentar custodias temporales, exhibiciones, devoluciones y responsabilidades asociadas.

# Prestamista

Definición.
Persona o entidad que entrega temporalmente una pieza al museo mediante préstamo.

Uso dentro de Instituva.
Se utiliza para identificar a quien conserva titularidad o derecho sobre la pieza prestada y para documentar las condiciones del préstamo.

# Depósito

Definición.
Condición en la que una pieza queda bajo custodia del museo sin que necesariamente exista transferencia de titularidad.

Uso dentro de Instituva.
Se utiliza para registrar piezas conservadas, resguardadas o administradas temporalmente por el museo bajo autorización.

# Ubicación Museográfica

Definición.
Lugar oficial donde se encuentra una pieza dentro del contexto museográfico o de custodia del museo.

Uso dentro de Instituva.
Se utiliza para localizar piezas en salas, áreas de exhibición, espacios de resguardo o cualquier lugar autorizado.

Notas.
No debe confundirse con ubicación administrativa de equipos operacionales.

# Movimiento

Definición.
Cambio documentado en la ubicación, custodia, estado operativo o contexto museográfico de una pieza.

Uso dentro de Instituva.
Se utiliza para preservar trazabilidad cada vez que una pieza cambia de lugar, condición de uso o responsabilidad.

# Traslado

Definición.
Movimiento físico de una pieza de un lugar a otro.

Uso dentro de Instituva.
Se utiliza cuando el cambio implica desplazamiento físico documentado de una pieza.

# Exhibición

Definición.
Presentación autorizada de una pieza en un contexto museográfico, educativo, cultural o institucional.

Uso dentro de Instituva.
Se utiliza para relacionar piezas con salas, eventos, muestras o programas expositivos.

# Condición

Definición.
Estado físico o de conservación observable de una pieza en un momento determinado.

Uso dentro de Instituva.
Se utiliza para documentar cómo se encuentra la pieza al ingresar, moverse, exhibirse, conservarse, restaurarse, prestarse o devolverse.

# Conservación

Definición.
Cuidado preventivo y seguimiento orientado a preservar la integridad de una pieza.

Uso dentro de Instituva.
Se utiliza para registrar acciones, observaciones o evaluaciones que buscan mantener la pieza en condiciones adecuadas.

# Restauración

Definición.
Intervención documentada orientada a estabilizar, recuperar o mejorar la condición de una pieza.

Uso dentro de Instituva.
Se utiliza para registrar procesos especializados aplicados a una pieza y su resultado institucional.

# Documento Asociado

Definición.
Documento que respalda, explica o evidencia información relacionada con una pieza o con un proceso de su ciclo de vida.

Uso dentro de Instituva.
Se utiliza para relacionar formularios, comunicaciones, autorizaciones, recibos, informes y documentos institucionales con el expediente museográfico.

# Evidencia Fotográfica

Definición.
Imagen que documenta la identidad, condición, ubicación, proceso o evento relacionado con una pieza.

Uso dentro de Instituva.
Se utiliza para respaldar el expediente museográfico y preservar evidencia visual de la pieza.

# Expediente Histórico

Definición.
Conjunto de información preservada sobre eventos pasados de una pieza o proceso museográfico.

Uso dentro de Instituva.
Se utiliza para consultar la trayectoria institucional de una pieza, incluso cuando un proceso ya concluyó.

# Valor Estimado

Definición.
Valor administrativo o museográfico aproximado asignado a una pieza para fines internos, documentales, de custodia, préstamo o evaluación.

Uso dentro de Instituva.
Se utiliza como referencia operacional dentro del expediente museográfico.

Notas.
No sustituye la contabilidad oficial.

# Valor Asegurado

Definición.
Valor administrativo utilizado como referencia para protección, cobertura o gestión de riesgo de una pieza.

Uso dentro de Instituva.
Se utiliza para documentar información necesaria para custodiar o proteger una pieza.

Notas.
No sustituye la contabilidad oficial.

# Valor de Reposición

Definición.
Valor administrativo utilizado para estimar el costo o referencia necesaria ante pérdida, daño o sustitución conceptual de una pieza.

Uso dentro de Instituva.
Se utiliza como información museográfica u operacional para gestión interna y protección institucional.

Notas.
No sustituye la contabilidad oficial.

# Pieza Activa

Definición.
Pieza cuyo expediente museográfico permanece vigente dentro de Colecciones.

Uso dentro de Instituva.
Se utiliza para identificar piezas disponibles para consulta, seguimiento, ubicación, conservación, exhibición o procesos museográficos activos.

# Pieza Prestada

Definición.
Pieza que se encuentra bajo préstamo documentado, ya sea recibida por el museo o prestada temporalmente por el museo.

Uso dentro de Instituva.
Se utiliza para distinguir piezas que requieren seguimiento de condiciones, fechas, responsabilidad y devolución.

# Pieza Devuelta

Definición.
Pieza cuyo proceso de préstamo o custodia temporal concluyó mediante devolución documentada.

Uso dentro de Instituva.
Se utiliza para preservar el cierre de un proceso temporal sin perder el historial asociado.

# Pieza en Conservación

Definición.
Pieza que se encuentra bajo seguimiento o cuidado preventivo documentado.

Uso dentro de Instituva.
Se utiliza para identificar piezas con atención de conservación activa o pendiente.

# Pieza en Restauración

Definición.
Pieza que se encuentra en un proceso documentado de restauración.

Uso dentro de Instituva.
Se utiliza para identificar piezas intervenidas o pendientes de resultado restaurativo.

# Baja Museográfica

Definición.
Salida formal de una pieza del inventario museográfico activo mediante proceso institucional documentado.

Uso dentro de Instituva.
Se utiliza para preservar el cierre o retiro de una pieza sin eliminar su memoria institucional.

# Expediente Cerrado

Definición.
Expediente museográfico que conserva su historial, pero cuyo ciclo activo concluyó administrativamente.

Uso dentro de Instituva.
Se utiliza para consultar información histórica de una pieza o proceso que ya no está activo.

# Inventario Museográfico

Definición.
Conjunto oficial de piezas registradas por Colecciones como patrimonio museológico o bajo responsabilidad museográfica del museo.

Uso dentro de Instituva.
Se utiliza para consultar, organizar y reportar las piezas reconocidas dentro del dominio Colecciones.

Notas.
No es sinónimo de Inventario Administrativo.

# Reporte Museográfico

Definición.
Presentación organizada de información relacionada con piezas, expedientes, procesos o estado del patrimonio museológico.

Uso dentro de Instituva.
Se utiliza para comunicar información de Colecciones a usuarios, dirección, administración, auditoría o áreas relacionadas.

# Historial

Definición.
Secuencia preservada de eventos, cambios y procesos relacionados con una pieza o expediente.

Uso dentro de Instituva.
Se utiliza para mantener trazabilidad institucional y evitar pérdida de memoria museográfica.

# Auditoría

Definición.
Revisión formal de información, decisiones, cambios o procesos para confirmar integridad, trazabilidad y responsabilidad institucional.

Uso dentro de Instituva.
Se utiliza para validar que la información del sistema conserva coherencia, evidencia y control institucional.

# Responsable

Definición.
Persona asignada a una tarea, proceso, expediente o acción dentro del sistema.

Uso dentro de Instituva.
Se utiliza para identificar quién tiene responsabilidad administrativa u operacional sobre una acción determinada.

# Custodio

Definición.
Persona, área o entidad encargada del cuidado o resguardo de una pieza durante un periodo o condición específica.

Uso dentro de Instituva.
Se utiliza para documentar responsabilidad de custodia sin asumir necesariamente titularidad.

# Investigador

Definición.
Persona que consulta, estudia o documenta información relacionada con una pieza o conjunto museográfico.

Uso dentro de Instituva.
Se utiliza para identificar participación en procesos de investigación, documentación o interpretación museográfica.

# Reglas del Lenguaje Oficial

- Un concepto tiene un solo nombre.
- Un nombre representa un solo concepto.
- Ningún documento futuro deberá redefinir estos conceptos.
- Ningún módulo deberá utilizar terminología distinta para el mismo concepto.
- Siempre que sea posible, el código deberá reflejar este lenguaje.
- La ayuda contextual integrada del sistema deberá derivarse de estas definiciones.
- Antes de introducir un término nuevo en el proyecto deberá verificarse que no exista ya una definición oficial en este documento.
- Si un término nuevo resulta necesario, deberá incorporarse primero a este documento antes de utilizarse en la arquitectura, el código o la interfaz.
